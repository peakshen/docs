# 2024年11月更新记录

**更新时间： 2024年11月4日**

**更新内容：**

**新功能：**

1. SecretPad建模画布支持定时周期任务创建、查看和管理 ，可按分区读取 odps 数据源 
2. 数据新增 MySQL 数据源，支持多样化场景数据接入
3. 画布组件库新增读模型、模型回测算子
4. 新增了 Prometheus 常规监控指标，默认关闭
5. 数据按字段授权到项目优化
