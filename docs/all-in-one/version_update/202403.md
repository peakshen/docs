# 2024年3月更新记录

**更新时间：** 2024年3月23日

**更新内容：**

**新功能：**

1、支持模型在线预测；

2、增加模型管理模块，支持查看打包的模型列表；

3、增加 http 数据源，并支持模型发布关联 http 数据进行离在线特征映射；

**功能更新：**

1、广义线性回归模型性能优化、增加分箱修改、模型预测组件优化、隐私求交组件支持left join、full join等；

2、日志优化，数据同步优化降低资源消耗；
