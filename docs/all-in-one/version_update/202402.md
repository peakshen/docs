# 2024年2月更新记录

**更新时间：** 2024年2月4日

**更新内容：**

**新功能：**

1、增加以下特征预处理组件：二元操作、交叉决策、特征计算、异常值填充、onehot_encode、常规分箱；

2、增加以下数据过滤组件：行级过滤；

3、增加以下统计组件：分组统计；

**功能更新：**

1、重新划分了组件类型；

2、对于所有的特征预处理组件（除分箱），用统一的特征工程应用组件进行规则应用；

在P2P和中心化部署模式下均支持以上组件。
